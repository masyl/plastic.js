    7# Why?
    8
    9Why use long function chains for simple math operations? But before

    14math with function if your already chaining methods. Ex.:
    15 var maxSizeInK = conf("maxSize").dividedBy(1024).round();

    17Second, we have a templating engine called "Dali" which will soon get a
    18rewrite. In its new version this templating engine will not allow the

    20full-fledge expressions.
    21
    22But wait, There is a third scenario which is not yet implemented. While
    23operations on simple number like "3 + 5" is easy to do with standard operators,
    24operation of arrays and matrices are really complexe. So, eventually this api
